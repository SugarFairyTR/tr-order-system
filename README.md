# 🚀 주식회사 티알코리아 주문 자동화 시스템

## 📋 프로젝트 개요

외근 업무 중 실시간 주문 처리를 위한 **Progressive Web App (PWA)** 기반 주문 자동화 시스템입니다.
모바일 최적화와 오프라인 기능을 지원하여 언제 어디서나 편리하게 주문을 관리할 수 있습니다.

## ✨ 주요 기능

### 🔐 **보안 시스템**
- PIN 기반 사용자 인증
- 담당자별 접근 제어
- 세션 관리 및 로그아웃 기능

### 📱 **모바일 최적화**
- 반응형 디자인으로 모든 디바이스 지원
- PWA 기능으로 앱처럼 설치 가능
- 터치 친화적 인터페이스

### 💾 **스마트 데이터 관리**
- **localStorage 우선 저장**: 즉시 저장, 번거로움 없음
- **선택적 파일 백업**: 원할 때만 파일로 저장
- **자동 백업**: 30분마다 자동 백업
- **데이터 무결성 검사**: 손상된 데이터 자동 복구
- **오프라인 동기화**: 네트워크 복구 시 자동 동기화

### 🔍 **고급 검색 및 필터링**
- **실시간 검색**: 주문번호, 담당자, 판매처, 품목 등 통합 검색
- **다중 필터**: 담당자, 날짜, 과거 주문 표시 옵션
- **디바운싱**: 성능 최적화된 검색

### 📊 **통계 분석**
- **실시간 통계**: 총 주문수, 총 매출, 평균 주문금액
- **담당자별/카테고리별/월별 분석**
- **최다 주문/최고 매출 판매처 분석**
- **시각적 통계 대시보드**

### 🛡️ **고급 에러 처리**
- **사용자 친화적 에러 메시지**
- **자동 오류 로그 수집**
- **손상된 데이터 자동 복구**
- **네트워크 오류 감지 및 처리**

### ⚡ **성능 최적화**
- **디바운싱**: 입력 지연 처리로 성능 향상
- **메모리 관리**: 이벤트 리스너 자동 정리
- **GPU 가속**: 부드러운 애니메이션
- **지연 로딩**: 필요할 때만 데이터 로드

## 🗂️ 파일 구조

```
📦 주문자동화시스템/
├── 📄 index.html              # 메인 HTML 파일
├── 🎨 styles.css              # 스타일시트
├── ⚡ app.js                  # 메인 애플리케이션 로직
├── 🔧 sw.js                   # Service Worker (PWA)
├── 📱 manifest.json           # PWA 매니페스트
├── 👥 user_config.json        # 사용자 설정 파일
├── 🗄️ database_optimized.json # 최적화된 데이터베이스
├── 🗄️ database_converted.json # 레거시 데이터베이스
└── 📖 README.md               # 프로젝트 문서
```

## 🚀 시작하기

### 1️⃣ **파일 설정**
```bash
# 모든 파일을 웹 서버 디렉토리에 복사
# (예: Apache htdocs, Nginx html 등)
```

### 2️⃣ **사용자 설정**
`user_config.json`에서 담당자 추가/수정:
```json
{
  "users": {
    "홍길동": {
      "pin": "1234",
      "name": "홍길동",
      "role": "영업사원",
      "email": "hong@company.com"
    }
  }
}
```

### 3️⃣ **데이터베이스 설정**
- `database_optimized.json`: 최신 최적화된 데이터
- `database_converted.json`: 레거시 지원용

### 4️⃣ **웹 서버 실행**
```bash
# Python 개발 서버 (테스트용)
python -m http.server 8000

# 또는 Node.js 서버
npx serve .
```

### 5️⃣ **브라우저에서 접속**
```
http://localhost:8000
```

## 🎯 사용 방법

### 🔑 **로그인**
1. 담당자 선택
2. 4자리 PIN 입력
3. 로그인 완료

### 📝 **주문 입력**
1. **담당자** → **판매처** → **도착지** 순서로 선택
2. **분류** → **품목** 선택
3. **수량, 단가, 도착일, 도착시간** 입력
4. **저장** 버튼 클릭
5. 선택적으로 **파일 백업** 실행

### 🔍 **주문 검색**
- 검색창에 키워드 입력 (주문번호, 담당자, 판매처 등)
- 담당자/날짜 필터 적용
- 과거 주문 표시 옵션 활용

### 📊 **통계 확인**
- 주문 목록 화면에서 **통계** 버튼 클릭
- 실시간 매출 및 주문 분석 확인

### ✏️ **주문 수정/삭제**
1. **수정** 탭에서 주문 선택
2. **수정하기** 또는 **삭제** 실행

## 🆘 문제 해결

### **일반적인 문제**
```bash
# 데이터가 표시되지 않는 경우
1. F12 → Console 탭에서 오류 확인
2. 페이지 새로고침 (Ctrl+F5)
3. 브라우저 캐시 삭제

# 파일 저장이 안 되는 경우
1. Chrome/Edge 브라우저 사용 권장
2. 팝업 차단 해제
3. 다운로드 권한 확인

# 오프라인에서 작동하지 않는 경우
1. HTTPS 연결 확인
2. Service Worker 등록 확인
3. 브라우저 저장 공간 확인
```

### **데이터 복구**
```javascript
// 브라우저 Console에서 실행
// 백업 데이터 확인
console.log(localStorage.getItem('trkorea_orders_backup'));

// 수동 데이터 복구
app.repairData();

// 데이터 무결성 검사
app.validateDataIntegrity();
```
