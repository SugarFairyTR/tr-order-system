# 🚀 티알코리아 주문시스템 V3.0

## 📋 완전 새로운 시작

**버전 3.0**은 기존 시스템의 모든 문제점을 해결하고 처음부터 새롭게 작성된 **완전히 새로운 주문 관리 시스템**입니다.

### 🎯 V3.0의 핵심 개선사항

- ✅ **단순하고 명확한 코드 구조** - 더 이상 복잡하지 않습니다
- ✅ **안정적인 로그인 시스템** - PIN 기반 보안 인증
- ✅ **Firebase 자동 저장** - 버튼 한 번으로 클라우드 동기화
- ✅ **완벽한 모바일 최적화** - 안드로이드 기기에서 완벽 작동
- ✅ **직관적인 사용자 인터페이스** - 터치 친화적 디자인
- ✅ **오프라인 지원** - 네트워크 없이도 작업 가능

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Backend**: Firebase Realtime Database
- **PWA**: Service Worker, Web App Manifest
- **스타일**: CSS Grid, Flexbox, CSS Variables
- **아이콘**: Font Awesome 6.0

## 🚀 빠른 시작

### 1️⃣ 파일 설치
```bash
# 모든 파일을 웹 서버 디렉토리에 복사
cp -r 117_WEB_AUTOMATION_V3.0/* /var/www/html/
```

### 2️⃣ Firebase 설정 (선택사항)
1. [Firebase Console](https://console.firebase.google.com/) 접속
2. 새 프로젝트 생성
3. Realtime Database 활성화
4. 웹앱 추가 후 설정 복사
5. `firebase-config.json` 파일 업데이트

### 3️⃣ 웹 서버 실행
```bash
# Python 개발 서버
python -m http.server 8000

# 또는 Node.js
npx serve .
```

### 4️⃣ 브라우저에서 접속
```
http://localhost:8000
```

## 📱 사용 방법

### 🔑 로그인
1. 담당자 선택
2. 4자리 PIN 입력
3. 로그인 버튼 클릭

### 📝 주문 입력
1. **담당자 선택** → 자동으로 해당 담당자의 판매처 목록 표시
2. **판매처 선택** → 자동으로 해당 판매처의 도착지 표시
3. **도착지 선택**
4. **분류 선택** → 자동으로 해당 분류의 품목 표시
5. **품목 선택**
6. **수량, 단가 입력** → 실시간 총액 계산
7. **도착일, 도착시간 선택**
8. **💾 저장 버튼 클릭** → Firebase + 로컬 저장

### 📋 주문 목록 확인
- 하단 네비게이션에서 **목록보기** 클릭
- 최신 순으로 정렬된 주문 목록 확인
- 각 주문의 상세 정보 확인

## 🐛 문제 해결

### 로그인 안됨
```bash
1. F12 → Console 확인
2. user_config.json 파일 존재 확인
3. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
```

### Firebase 연결 안됨
```bash
1. firebase-config.json 파일 확인
2. Firebase 프로젝트 설정 재확인
3. 네트워크 연결 상태 확인
```

### 데이터 표시 안됨
```bash
1. database_optimized.json 파일 확인
2. 브라우저 새로고침 (Ctrl+F5)
3. localStorage 초기화
```
