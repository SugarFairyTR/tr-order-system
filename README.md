# 📱 티알코리아 주문 시스템

## 🏢 회사 정보
- **회사명**: 주식회사 티알코리아
- **업종**: 설탕/첨가물 수입업
- **담당자**: 김정진, 박경범, 이선화, 신준호

## 📋 시스템 소개
영업사원들이 외근 중 스마트폰으로 주문을 입력할 수 있는 모바일 웹앱입니다.

### ✨ 주요 기능
- 👤 담당자별 PIN 로그인
- 🏢 판매처/도착지 관리
- 📦 설탕/식품첨가물 분류
- 💰 가격 자동계산
- 📱 PWA 지원 (앱처럼 설치 가능)
- 🔒 보안 시스템 (1시간 자동 로그아웃)

### 🔐 로그인 정보
각 담당자별 4자리 PIN 번호로 로그인

### 📱 사용 방법
1. 웹사이트 접속
2. 담당자 선택
3. PIN 번호 입력
4. "홈 화면에 추가"로 앱 설치
5. 주문 입력

## 🔧 기술 스택
- HTML5, CSS3, JavaScript
- Progressive Web App (PWA)
- Service Worker (오프라인 지원)
- JSON 데이터베이스
